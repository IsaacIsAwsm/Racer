# Racing Game

A fast-paced 3D racing game built with Three.js featuring split-screen multiplayer support.

## Features

- **1-4 Player Support**: Choose from 1 to 4 players with dynamic split-screen
- **Speed Powerups**: Collect pink powerups for 3-second speed boosts
- **6-Lap Races**: First to complete 6 laps wins
- **Racing Atmosphere**: Grandstands with fans and checkered flag finish line
- **Responsive Controls**: Keyboard controls for all players

## Controls

- **Player 1**: Arrow Keys (↑ accelerate, ← → steer)
- **Player 2**: WASD (W accelerate, A D steer)
- **Player 3**: IJKL (I accelerate, J L steer)
- **Player 4**: Numpad (8 accelerate, 4 6 steer)

## How to Play

1. Open `index.html` in any modern web browser
2. Select number of players (1-4)
3. Use your assigned keys to accelerate and steer
4. Collect pink powerups for speed boosts
5. Complete 6 laps to win!

## Technical Details

- Built with Three.js for 3D graphics
- Frontend-only - no server required
- Runs entirely in the browser
- Responsive design with dynamic viewport splitting

## Installation

Simply download the files and open `index.html` in your browser. No additional setup required!
